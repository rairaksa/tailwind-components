
# Tailwind Component Pack

A collection component of html built with tailwindcss

## Installation

Install tailwind

```bash
  $ npm install -D tailwindcss
```
Install live-server

```bash
  $ npm install -g live-server
```
    
## Run Locally

Start the server

```bash
  $ live-server
```
